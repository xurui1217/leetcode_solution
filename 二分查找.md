## 69 x的平方根

限定范围，左和右，然后依次找中间

中间小了，就左边=中间

中间大了，就右边=中间

``` python
class Solution:
    def mySqrt(self, x: int) -> int:
        # 为了照顾到 0 把左边界设置为 0
        left = 0
        # 为了照顾到 1 把右边界设置为 x // 2 + 1
        right = x // 2 + 1
        while left < right:
            # 注意：这里一定取右中位数，如果取左中位数，代码可能会进入死循环
            # mid = left + (right - left + 1) // 2
            mid = (left + right + 1) >> 1
            square = mid * mid

            if square > x:
                right = mid - 1
            else:
                left = mid
        # 因为一定存在，因此无需后处理
        return left
```

## 167 两数之和II，输入有序数组

双指针很简单

``` python
class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        n=len(numbers)
        l=0
        r=n-1
        while l<r:
            if numbers[l]+numbers[r]==target:
                return [l+1,r+1]
            if numbers[l]+numbers[r]<target:
                l+=1
            elif numbers[l]+numbers[r]>target:
                r-=1
        return None
```

## 278 第一个错误的版本

同样，双指针，判断左右边界

边界条件还是需要判断一下，尤其是一开始的时候需要判断一下1是不是True的问题。

``` python
# The isBadVersion API is already defined for you.
# @param version, an integer
# @return a bool
# def isBadVersion(version):

class Solution:
    def firstBadVersion(self, n):
        """
        :type n: int
        :rtype: int
        """
        if isBadVersion(1)==True:
            return 1
        l=1
        r=n
        while l<r:
            mid=l+(r-l+1)//2
            if r-l==1:
                return r
            if isBadVersion(mid)==False:
                l=mid
            elif isBadVersion(mid)==True:
                r=mid
        return None
```

349 两个数组的交集

技巧，转set，然后用set的交集直接算出来

```python 
class Solution:

    def intersection(self, nums1: List[int], nums2: List[int]) -> List[int]:
        set1=set(nums1)
        set2=set(nums2)
        set_res=set1 & set2
        res=list(set_res)
        return res

```
